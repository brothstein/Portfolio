Projects

  • Project 1

    This project was for my 'Mobile Development' class, in which I made an app that includes multiple screens and components. I utilized React Native, with React Native Elements and React Navigation, in this project. The final app is a simple recipe app, where a list of recipe cards are shown to the user, and can be selected to view more information (ingredients and instructions), and the user can choose to save the recipe. On the home page, the user can also filter the recipes shown by ingredients used or meal type. After saving a recipe, the recipe cards are shown on the saved recipes page. The GitHub Pages site for this project is at: https://brothstein.github.io/recipeapp/. The GitHub repository is availible at: https://github.com/brothstein/recipeapp.
